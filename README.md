## âœ¨ Features

### Core CRM
- **Account Management** - Organize and track companies/organizations
- **Contact Management** - Manage individual contacts linked to accounts
- **Lead Management** - Qualify and track sales leads
- **Opportunity Pipeline** - Manage sales opportunities with customizable stages

- **Invoice Management** - Create and track invoices with AI-powered data extraction

- **Knowledge Base** - Notion integration for team documentation
- **AI Insights** - OpenAI-powered summaries and automated notifications
- **Multi-Language** - English, German, Czech, Ukrainian
- **Dark Mode** - Complete dark theme support
- **Responsive Design** - Works on desktop, tablet, and mobile
- **Role-Based Access** - Admin and account admin roles

## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+ (recommend 20+)
- pnpm (or npm/yarn)
- MongoDB Atlas account
- Git

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/DrivenIdeaLab/nextcrm-app.git
   cd nextcrm-app
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env.local
   # Edit .env.local with your configuration
   ```

4. **Initialize database**
   ```bash
   pnpm run build
   ```
   This command will:
   - Generate Prisma client
   - Push schema to MongoDB
   - Seed initial data

5. **Start development server**
   ```bash
   pnpm run dev
   ```

The application will be available at `http://localhost:3000`

### Available Scripts
```bash
pnpm run dev     # Start development server
pnpm run build   # Build for production
pnpm run start   # Start production server
pnpm run lint    # Run ESLint
```

**Optional Services:**
- `OPENAI_API_KEY` - For AI features
- `RESEND_API_KEY` - For email delivery
- `NOTION_API_KEY` - For knowledge base
- OAuth provider credentials (Google, GitHub, Twitter)
- File storage (UploadThing or S3)

### Docker Installation

For self-hosted deployments, you can use Docker:

```bash
# Build image
docker build -t nextcrm .

# Run container
docker run -e DATABASE_URL="..." -p 3000:3000 nextcrm
```

See [Docker Hub](https://hub.docker.com/repository/docker/nextcrmio/nextcrm/general) for more details.

### Deployment Options

**Vercel (Recommended):**
- Easy setup with git integration
- Automatic deployments
- Built-in analytics and monitoring

**Self-Hosted:**
- Full control over infrastructure
- Docker containerization support
- Support for custom domains and SSL

See [ARCHITECTURE.md](ARCHITECTURE.md#deployment-architecture) for detailed deployment instructions.
