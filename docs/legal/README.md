## Data Breach Response

### If You Discover a Breach
1. **Contact immediately:** privacy@nextcrm.io
2. **Include:**
   - Type of breach
   - Data affected
   - Number of users impacted
   - When discovered
3. **Timeline:**
   - We respond: Within 24 hours
   - We investigate: Within 5 days
   - We notify authorities: Within 72 hours (GDPR)
   - We notify users: Without undue delay

### Our Commitment
- Immediate incident response
- Transparent communication
- Regulatory notification
- Preventative measures
- Regular security audits

## Request Process

### Data Subject Rights Requests

**How to request:**
1. Email: privacy@nextcrm.io
2. Include:
   - Full name
   - Email address
   - Organization
   - Type of request (access, delete, correct, export, etc.)
3. Response time: 30 days (GDPR requirement)

### DPA Request

**How to request:**
1. Email: legal@nextcrm.io
2. Include:
   - Organization name
   - Purpose (for signing)
3. Response time: 5 business days

### Security & Compliance Documents

**How to request:**
1. Email: legal@nextcrm.io
2. Specify which documents:
   - SOC 2 audit report
   - Penetration test results
   - Vulnerability assessment
   - ISO 27001 documents (in progress)
3. Response time: 5 business days
