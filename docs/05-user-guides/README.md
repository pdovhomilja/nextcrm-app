## Learning Paths

Choose your path based on your role:

### Path 1: "I'm New to NextCRM"
**Goal:** Get set up and productive in 30 minutes

1. Start: [Getting Started](./GETTING_STARTED.md)
   - Create your account
   - Set up your profile
   - Understand key concepts
   - Invite your team

2. Explore: [CRM Guide](./CRM-GUIDE.md)
   - Create your first account
   - Add contacts
   - Manage leads
   - Track opportunities

3. Reference: [Quick Start](./QUICK_START.md)
   - Keyboard shortcuts
   - Common workflows
   - Getting help

**Next Steps:** Explore module guides based on your role

---

### Path 2: "I'm Managing Projects"
**Goal:** Master project management and task tracking

1. Foundation: [Getting Started](./GETTING_STARTED.md)
   - Profile setup (5 min)

2. Core: [Projects Guide](./PROJECTS-GUIDE.md)
   - Create projects
   - Build Kanban boards
   - Assign tasks
   - Track progress
   - Use AI features

3. Administration: [Admin Guide](./ADMIN-GUIDE.md)
   - Manage team members
   - Set permissions
   - Configure settings

4. Advanced: [Permissions Guide](./PERMISSIONS-GUIDE.md)
   - Control access
   - Manage visibility

**Common Tasks:**
- Creating a project board
- Assigning tasks to team members
- Tracking project progress
- Using AI task suggestions

---

### Path 3: "I'm in Finance/Accounting"
**Goal:** Master invoicing, payments, and financial tracking

1. Quick Setup: [Getting Started](./GETTING_STARTED.md) (sections 1-3 only)

2. Core: [Invoices Guide](./INVOICES-GUIDE.md)
   - Create invoices
   - Upload invoice documents
   - Use Rossum AI extraction
   - Track payments
   - Generate reports

3. Supporting: [Documents Guide](./DOCUMENTS-GUIDE.md)
   - Organize invoice documents
   - Share with team
   - Archive old records

4. Reference: [CRM Guide](./CRM-GUIDE.md)
   - Link invoices to accounts
   - View customer history

**Common Tasks:**
- Processing supplier invoices
- Creating customer invoices
- Tracking overdue payments
- Generating revenue reports

---

### Path 4: "I'm a Sales Professional"
**Goal:** Master lead management, pipeline tracking, and customer relationships

1. Setup: [Getting Started](./GETTING_STARTED.md)
   - Profile & preferences (5 min)

2. Core: [CRM Guide](./CRM-GUIDE.md)
   - Create and qualify leads
   - Manage accounts & contacts
   - Track opportunities
   - Move deals through pipeline
   - Forecast revenue

3. Communication: [Email Guide](./EMAIL-GUIDE.md)
   - Connect email account
   - Use email templates
   - Track opens & clicks
   - Schedule follow-ups

4. Organization: [Documents Guide](./DOCUMENTS-GUIDE.md)
   - Store proposals
   - Share presentations
   - Manage contracts

**Common Tasks:**
- Importing leads from trade show
- Moving opportunity to next stage
- Scheduling follow-up emails
- Forecasting quarterly revenue

---

### Path 5: "I'm an Administrator"
**Goal:** Configure and manage your organization's NextCRM instance

1. Foundation: [Getting Started](./GETTING_STARTED.md)
   - Complete setup

2. Core: [Admin Guide](./ADMIN-GUIDE.md)
   - Organization settings
   - User management
   - Module configuration
   - Billing & subscriptions
   - Audit logs

3. Security: [Permissions Guide](./PERMISSIONS-GUIDE.md)
   - Understand roles (Owner, Admin, Member, Viewer)
   - Configure access control
   - Best practices

4. Reference: [FAQ](./FAQ.md) + [Troubleshooting](./TROUBLESHOOTING.md)
   - Support common issues
   - Help users get unstuck

**Common Tasks:**
- Adding new team members
- Changing user roles
- Reviewing audit logs
- Managing subscription
- Configuring modules
